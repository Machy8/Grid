# Resets

A few useful resets classes.

- **Border:** border-none
- **Padding:** padding-none
- **Margin:** margin-none
- **Background:** background-none
- **Outline:** outline-none
- **Shadow:** shadow-none
- **Width:** width-auto

## Examples
````Html
<input type="button" value="Click me!" class="border-none padding-none background-none">
````
