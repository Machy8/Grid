# Buttons
**By color**
- **Without color** - button
- **Green** - button-green
- **Orange** - button-orange
- **Red** - button-red

**By size**
- **Small** - button-small
- **Small and wide** - button-small-wide
- **Wide** - button-wide
- **Large** - button-large
- **Large and wide** - button-large-wide
