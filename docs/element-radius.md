# Element radius

A few classes with which you can easily change the border-radius

- **No radius:** square
- **Rounded:** rounded-(1-4 always increased by 2)
- **Circle:** circle

## Examples
````Html
<!-- Red circle -->
<div class="circle" style="width: 20px; height: 20px; background-color: blue;">
 Some content
</div>

<!-- border-radius: 4 -->
<div class="rounded-2">
 Some content
</div>
````
