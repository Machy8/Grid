# Visibility

You can hide element using display types or by changing its opacity.

- **Transparent:** transparent
- **Transparency:** transparency-(1-10)
- **Opacity 1:** opaque

##Examples
````Html
<!-- opacity: 0.2 -->
<div class="transparency-2">
Some content
</div>

<!-- opacity: 1 -->
<div class="Opaque">
Some content
</div>
````
