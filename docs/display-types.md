# Display types
All display properties are available with and without prefixes

- **Block:** display-block
- **Inline:** display-inline
- **Inline block:** display-inline-block
- **None:** display-none | hidden |
- **Flex:** display-flex

## Examples
````Html
<!-- Alias for s-block -->
<div class="display-block">
Some content
</div>

<div class="s-block">
Some content
</div>

<div class="s-display-block">
Some content
</div>
````
